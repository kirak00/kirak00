모든 자료가 성공적으로 등록 되었습니다.

신청하신 대출 비교 건에 대해 금융 기관의 제안서가 도착하면 아래 기준으로 열람이 가능합니다.

금융기관별 내부 심사일에 따라 고객님이 등록하신 제안서 마감일까지 제안서가 제출될 수 있습니다.

신용대출
· 대출 신청 금액 1억 미만 : 대출 비교 하기 일 + 1일 부터 확인 가능
· 대출 신청 금액 1억 이상 : 대출 비교 하기 일 + 3일 부터 확인 가능
담보대출
· 대출 신청 금액 5억 미만 : 대출 비교 하기 일 + 5일 부터 확인 가능
· 대출 신청 금액 5억 이상 : 대출 비교 하기 일 + 15일 부터 확인 가능

금융기관이 심사 중 필요한 경우 유선 연락 또는 현장방문 요청을 받으실 수 있어요.
